# FCG Taskboard

Free open-source kanban task board with Google Calendar integration. Drag-and-drop task management with search, tag filtering, priority levels, due dates, and white-label configuration for any organization.

Built by [FCG-builds](https://github.com/FCG-builds) (Fairy Circle Garden).

## Features

- **Drag & drop** tasks between columns
- **Google Calendar sync** -- auto-load tasks from your calendar
- **Priority levels** -- high, medium, low with color coding
- **Tags** -- categorize and filter tasks
- **Search** -- find tasks instantly
- **Due dates** -- track deadlines
- **Notes** -- add detail to any task
- **Edit modal** -- click any task to edit all fields
- **Custom columns** -- add, remove, rename columns
- **White-label** -- set your org name in Settings
- **Persistent** -- saves to localStorage automatically
- **Dark theme** -- easy on the eyes
- **Zero dependencies** beyond React

## Quick Start

```bash
git clone https://github.com/FCG-builds/fcg-taskboard.git
cd fcg-taskboard
npm install
npm run dev
```

## For Your Business

Click the gear icon to set your organization name, add custom columns for your workflow, and reset the board. All data stays in your browser -- no accounts, no cloud, no tracking.

Fork this repo, customize the columns and branding, deploy to any static host.

## Tech Stack

- React 18 + Vite
- Zero external dependencies
- localStorage persistence
